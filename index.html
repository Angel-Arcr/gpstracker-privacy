Política de Privacidad

GPSTracker

1. Recopilación de datos

GPSTracker recopila datos de ubicación únicamente con el propósito de mostrar y registrar recorridos del usuario dentro de la aplicación.

2. Uso de la información

La información de ubicación se utiliza solamente para el funcionamiento principal de la aplicación, como calcular distancia, tiempo y visualizar recorridos.

3. Compartición de datos

GPSTracker no comparte, vende ni transfiere la información del usuario a terceros.

4. Control del usuario

El usuario puede permitir o desactivar el acceso a la ubicación en cualquier momento desde la configuración de permisos de su dispositivo Android.
