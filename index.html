Política de Privacidad - GPSTracker

Esta aplicación recopila datos de ubicación con el fin de mostrar y registrar recorridos del usuario.

La información de ubicación se utiliza únicamente dentro de la aplicación y no se comparte con terceros.

No se recopila información personal como nombre, correo o datos sensibles.

El usuario puede desactivar el acceso a la ubicación en cualquier momento desde la configuración de su dispositivo.
